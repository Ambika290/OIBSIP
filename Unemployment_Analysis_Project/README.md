# Unemployment Analysis with Python

## Oasis Infobyte Data Science Internship

### Project Overview

This project analyzes unemployment trends in India using Python and data visualization techniques. The objective is to understand unemployment patterns across different regions and compare unemployment rates in urban and rural areas.

### Dataset

The dataset contains:

* Region
* Date
* Estimated Unemployment Rate (%)
* Estimated Employed
* Estimated Labour Participation Rate (%)
* Area (Urban/Rural)

### Technologies Used

* Python
* Pandas
* Matplotlib

### Analysis Performed

* Data Cleaning and Preprocessing
* Unemployment Rate Distribution Analysis
* Average Unemployment Rate by Region
* Urban vs Rural Unemployment Comparison

### Visualizations

* Histogram of Unemployment Rate Distribution
* Bar Chart of Average Unemployment Rate by Region
* Pie Chart of Urban vs Rural Unemployment Rate

### Key Findings

* Unemployment rates vary significantly across different regions of India.
* Tripura and Haryana recorded higher unemployment rates.
* Urban unemployment was slightly higher than rural unemployment.
* Regional differences indicate varying employment opportunities across states.

### Project Outcome

The project provides valuable insights into unemployment trends in India through data analysis and visualization. These findings can help in understanding employment challenges and support data-driven decision-making.
