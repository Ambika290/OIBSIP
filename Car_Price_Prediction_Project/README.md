# Car Price Prediction with Machine Learning

## Project Overview

This project aims to predict the selling price of used cars using Machine Learning. Various car features such as manufacturing year, present price, kilometers driven, fuel type, transmission type, and ownership information are used to estimate the selling price.

## Objective

To build a machine learning model that accurately predicts the selling price of a car based on its characteristics.

## Dataset Description

The dataset contains information about used cars, including:

* Car_Name
* Year
* Selling_Price (Target Variable)
* Present_Price
* Driven_kms
* Fuel_Type
* Selling_type
* Transmission
* Owner

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Machine Learning Algorithm

* Random Forest Regressor

## Project Workflow

1. Data Loading
2. Data Preprocessing
3. Feature Engineering (Car Age Calculation)
4. Categorical Data Encoding
5. Train-Test Split
6. Model Training using Random Forest Regressor
7. Prediction and Evaluation

## Visualizations

* Selling Price Distribution Histogram
* Present Price vs Selling Price Scatter Plot

## Model Performance

* Mean Absolute Error (MAE): 0.6494
* R² Score: 0.9585

## Results

The Random Forest Regressor achieved excellent performance with an R² Score of 0.9585, indicating that the model can accurately predict car selling prices.

## Conclusion

Machine Learning can effectively predict used car prices based on vehicle characteristics. The developed model achieved high accuracy and can assist buyers and sellers in estimating fair vehicle prices.
