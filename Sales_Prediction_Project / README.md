# Sales Prediction Using Python

## Oasis Infobyte Data Science Internship

### Project Overview
This project predicts product sales based on advertising budgets spent on:
- TV
- Radio
- Newspaper

Machine Learning techniques are used to analyze the relationship between advertising expenditure and sales.

### Dataset
The dataset contains:
- TV Advertising Budget
- Radio Advertising Budget
- Newspaper Advertising Budget
- Sales

### Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

### Machine Learning Model
- Linear Regression

### Project Outcome
The model predicts future sales based on advertising investments and helps businesses make data-driven marketing decisions.
